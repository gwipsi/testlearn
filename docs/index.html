PowerShell Git Lernprojekt

Diese Datei ist eine einzelne HTML-Seite mit Inline-CSS und Inline-JavaScript. Sie ist absichtlich standalone, damit du sofort etwas Sichtbares im Browser hast.

HTML

Standalone-Datei

Alles steckt in einer Datei: Struktur, Design und Logik. Das ist fuer ein Lernprojekt praktisch, aber fuer groessere Projekte nicht die sauberste Architektur.

PowerShell

Skriptbezug

Das Dashboard zeigt beispielhaft, welche Informationen dein PowerShell-Skript ausgeben koennte: Computername, Benutzername und Zeitstempel.

Git

Update-Fluss

Diese Karte wurde in einer eigenen Branch geaendert. So sieht man im Pull Request exakt, welche Zeilen angepasst wurden.

Interaktive Demo

Klicke auf die Buttons. Die Ausgabe wird im Browser durch Inline-JavaScript erzeugt.

Bereit.
Lernhinweis: Inline-Code ist hier absichtlich erlaubt, weil das Ziel Sichtbarkeit und Einfachheit ist. Spaeter sollte CSS und JavaScript getrennt werden.
